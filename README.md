# MineWorld 3D

这是一个完全基于 HTML5、Three.js 和 Web Audio API 构建的 3D《我的世界》网页版克隆项目。
包含纯代码合成的正宗物理音效、程序化地形生成（无限挖掘）、以及真正的跨国 WebSocket 多人联机功能！

## 功能特点
- **完全方块化材质**：没有绝对的圆形，所有贴图（草地、原木、泥土）均为 16x16 像素噪点随机生成。
- **极致的无限挖掘**：采用动态表面剔除算法，向下挖 10000 层也不会卡顿。
- **纯粹的合成音效**：没有使用任何现成的 MP3 录音。泥土的“沙沙”声、石头的“咣咣”声、玻璃破碎的“叮当”声全部通过 Web Audio API 纯代码实时物理合成。
- **多人联机**：自带一个轻量级的 WebSocket 服务器，支持玩家输入公网 IP 进行真实的跨国联机。可以看到彼此的动作并同步破坏/放置方块。
- **丰富的机制**：包含核弹爆炸、枪械射线检测、火把供电的熔炉、村民建造彩蛋、以及水下窒息（20秒）等物理机制。

## 如何运行单人游戏
由于使用了本地文件系统和 `fetch`（或者本地资源），你不能直接双击打开 HTML 文件，需要启动一个本地服务器：

1. 确保你安装了 Python 3 或 Node.js
2. 在项目根目录下打开终端
3. 运行本地服务器：
   - Python: `python3 -m http.server 8000`
   - Node.js: `npx serve -p 8000`
4. 打开浏览器访问：[http://localhost:8000/game3d.html](http://localhost:8000/game3d.html)

## 如何开启多人联机服务器
如果你想和朋友联机，需要运行后端的 WebSocket 服务器。

1. 安装依赖 (需要 Node.js 或 Python 环境):
   如果你使用 Python (推荐):
   `pip install websockets`
   
2. 启动联机服务器:
   `python3 server.py`

3. 游戏内操作:
   - 在游戏中点击“联机游戏”。
   - 界面上会自动显示你的公网 IP 地址。
   - 把你的公网 IP 发给朋友。
   - 朋友在他们的游戏里点击“联机游戏”，输入你的 IP 并点击连接。

## 如何部署到云端让所有人都能玩 (全球联机)
如果你想让全世界的人都能玩你的游戏，并且能在全球各地联机（比如你在海南，朋友在俄罗斯），你需要把“游戏网页”和“联机服务器”分别部署到云端：

### 1. 网页部署 (前端 - 使用 GitHub Pages)
- 这是你已经创建的 GitHub 仓库 (`github.com`)。
- 在 GitHub 仓库的 **Settings** -> 左侧边栏的 **Pages** 中，将 Source 设置为 `main` 分支。
- 稍等片刻，你就能获得一个 `https://你的用户名.github.io/仓库名/game3d.html` 的网址。发给朋友，他们就能打开游戏。

### 2. 联机服务器部署 (后端 - 为什么不能只用 GitHub)
**注意**：GitHub (`github.com`) 本身只提供“代码存储”和“静态网页展示 (GitHub Pages)”，它**不允许**在上面运行长时间开启的后端程序（比如咱们用来做联机的 WebSocket 服务器 `server.py`）。

如果你只在 GitHub 上部署了网页，你和朋友确实都能打开游戏，但**无法跨国联机看到彼此**（只能单机玩）。

要实现真正的联机，你需要把 `server.py` 跑在一个能运行代码的云平台上（通常需要使用如 Render, Railway 等免费服务）。

#### 推荐方案：使用 Render
1. 打开并注册：[https://render.com/](https://render.com/)
2. 点击右上角 **New +**，选择 **Web Service**。
3. 关联你的这个 GitHub 仓库 (`MineWorld-3D`)。
4. 在设置中，找到 `Start Command`，输入：`python3 server.py`
5. 创建完毕后，Render 会给你分配一个专属的服务器地址（例如：`your-mc-server.onrender.com`）。

### 3. 最终联机
- 你和朋友都打开你部署在 GitHub 上的网页。
- 点击“联机游戏”。
- 在输入框里填入 Render 给你的那个专属服务器地址（例如：`your-mc-server.onrender.com`）。
- 点击连接，你们就成功在全球网络里汇合了！
